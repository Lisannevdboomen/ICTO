New: The UvA switched from Mediasite to Panopto

From Mediasite to Panopto
After the Christmas holidays, the web lecture service made a transition to a new supplier, resulting in the replacement of 35 recorders in the lecture halls. Starting from January 8th, lectures will no longer be recorded using Mediasite; instead, they will be recorded using the new system Panopto. The procedure for requesting web lectures will stay unchanged, and utilizing the new system is fairly straightforward.

Accessing old videos
In the meantime, new content can be accessed through Panopto, while old videos can still be viewed on Mediasite, and they will remain accessible until the migration is fully completed.

Technical issues
With the switch to Panopto, you may have encountered some initial challenges. Technical issues arose during the first week, but they have since been resolved. Unfortunately, there are also some limitations with Panopto, as it is currently not available through the rich content editor in Canvas, unlike Mediasite. This means that teachers can't easily access lectures and embed them as they could before. The web lecture service is working to address this issue, but it is uncertain when this option will be reinstated in the future.

Requesting web lectures in the planner

Teachers can request one or several recordings with http://webcollegeplanner.uva.nl/. If you have any questions about web lectures, please contact webcolleges@uva.nl (T: 020 525 2800).

What functionalities can I request with the webcollegeplanner?
Recording          →  A full video recording, visual and audio.
Livestream         →  A streaming link and recording.
Sound + Slides   →  Audio recording without video, but the desktop (powerpoint) included.
Forward             →  Allows you to stream to another room.

Please note:
→  If there's a change of location or room, the request for the recording (or livestream) must be submitted again via the webcollegeplanner.
→  Do not forget to wear the microphone provided by the webcollegedienst during the lecture, otherwise the weblecture recording will contain no audio and/or students online won't hear you.
→  The livestream function has a delay of about 30 seconds which makes it not ideal for communicating with students at home. Alternitavely, a BRIO device is attached in every lecture hall by default and this will allow you to achieve the same goal, only without delay. Email icto-csw@uva.nl if you have any questions about it.

Sharing your lectures with students

Share an URL link with students
Teachers can request one or several recordings with http://webcollegeplanner.uva.nl/. After requesting the lectures, you will receive a URL link to the lectures in Panopto. You can share this link with students.

Sharing the lectures with one or few students
Unfortunately, Panopto is currently not available through the rich content editor in Canvas, unlike Mediasite. This means that teachers can't easily access lectures and embed them as they could before. The web lecture service is working to address this issue, but it is uncertain when this option will be reinstated in the future. Please email icto-csw@uva.nl if you have questions about sharing lectures with only one or few students.
