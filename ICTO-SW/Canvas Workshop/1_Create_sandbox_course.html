Create your own sandbox course

  1. Go to this link: https://datanose.nl/#canvas[FMG]
  2. Log in using your UvAnetID



  3. Select Your sandbox course



  4. You can find your course on Canvas via Courses Unpublished courses

Maak je eigen testcursus

  1. Ga naar deze link: https://datanose.nl/#canvas[FMG]
  2. Log in met je UvAnetID



  3. Klik op Jouw testcursus



  4. Vind jouw testcursus op Canvas onder Courses Unpublished courses
