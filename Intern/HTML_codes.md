# Anchor link :anchor:
> Klik op een link & spring naar een andere plek in de pagina

:heavy_check_mark: ### Plain text anchor link

##### Bron
```html
<a href="#stap1">Inloggen in TestVision</a>
```
##### Destination/anchor
```html
<p>
    <a id="stap1"></a>
</p>
```
:heavy_check_mark: ### Button anchor link

##### Bron
```html
<p>
    <a class="btn Button" style="margin: 5px;" href="#topmenu">
        <span> Go back to menu </span>
    </a>
</p>
```
##### Destination/anchor
```html
<p>
    <a id="topmenu"></a>
</p>
