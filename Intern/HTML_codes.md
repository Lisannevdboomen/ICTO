# Anchor link
> Klik op een link & spring naar een andere plek in de pagina

## Plain text link

##### Bron/link
```html
<a href="#menu1">Inloggen in TestVision</a>
```
##### Destination/anchor
```html
<p>
    <a id="menu1"></a>
</p>
```
