Maak een opdracht aan (met plagiaatcheck)

  1. Verlaat de student view weer
  2. Ga in het linker menu naar Assignments
  3. Klik op + Assignment
  4. Vul een beschrijving van de opdracht in
  5. Kies een aantal punten (meestal is dit 10)
  6. Kies 'Online' als Submission Type en selecteer 'file uploads'
  7. Bij Plagiarism review selecteer je 'Turnitin Similarity'
  8. Bij Assign to kies je voor de sectie 'Everyone'
  9. Bij Due kies je een deadline

    Let op: studenten kunnen na deze deadline nog wel bestanden inleveren; deze worden als 'laat' gemarkeerd. We adviseren geen 'until' datum te gebruiken, omdat studenten na deze datum helemaal geen bestanden meer kunnen inleveren.

  10. Klik op Save & Publish
  11. Ga in het linker menu naar Modules
  12. Klik op rechts naast de module
  13. Kies in de lijst met assignments voor de opdracht die je zojuist hebt aangemaakt
  14. Klik op Add item

Create an assignment (with plagiarism review)

  1. Leave the student view
  2. Go to Assignments in the menu on the left
  3. Select + Assignment
  4. Enter a description of the assignment
  5. Choose a number of points (usually 10)
  6. Choose 'Online' as Submission Type and select 'file uploads'
  7. Under Plagiarism review select 'Turnitin Similarity'
  8. Under Assign to choose 'Everyone'
  9. Under Due you can choose a deadline

    Please note that students can still submit files after this deadline; these will be marked as 'late'. We do not recommend using a 'until' date because students cannot submit files at all after this date.

  10. Click on Save & Publish
  11. Go to Modules in the menu on the left
  12. Click next to the module
  13. In the list of assignments, choose the assignment you have just created
  14. Select Add item
