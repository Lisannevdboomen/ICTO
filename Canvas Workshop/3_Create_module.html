Maak een module aan

  1. Ga in het linker menu naar Modules
  2. Klik op + Module
  3. Vul de naam van jouw module (bijvoorbeeld: "Course information")
  4. Klik op Add module
  5. Klik op rechts naast de module
  6. Kies voor Page en klik vervolgens op Create page
  7. Vul de naam van jouw pagina (bijvoorbeeld: "Schedule")
  8. Klik op Add item
  9. Vergeet niet de module en alle bijbehorende onderdelen te publiceren door op het icoon te klikken

Create a module

  1. Go to Modules in the menu on the left
  2. Select + Module
  3. Add a title to the module (for example: "Course information")
  4. Select Add module
  5. Select on the right side of the module
  6. Choose Pageand click on Create page
  7. Add a title to the page (for example: "Schedule")
  8. Select Add item
