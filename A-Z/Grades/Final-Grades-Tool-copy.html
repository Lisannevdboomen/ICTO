About Final Grades

From '21-'22 the Final grades tool saves you a lot of time when submitting grades to SIS. This pages explains how to use it.

The instructions menu below takes you through the whole process. Beneath that is an instructional videoplaylist. ICTO Social Sciences will not be able to assist you with calculations that exceed the basic or advanced examples. For these instances, set up the calculation and use the Test calculation feature.

How to access:   Final Grades menu   Grade calculation setup

Examples

Basic
  • 4 Canvas assignments that are each worth a maximum of 10 points.
  • These assignments need to have their own mutual weight of 3 x 20% and 1 x 40%
  View basic example
Advanced
  • Canvas Exams (i.e. Canvas Quizzes) is included in this setup
  • These quizzes are worth 15 points in total so they need to be scaled to a grade of 10 points
  • The lowest grade will be dropped
  • The quizzes are in two versions: one English version and one Dutch
  View advanced example

More options

You can also combine items together into a group to release certain conditions or calculations. First create a so-called "calculated item". It serves as a folder in which you place several items together and can be recognized in Final Grades by a blue link (whereas regular items are displayed in plain text).
