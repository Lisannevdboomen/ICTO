About the self-enrollment system

It is possible to have the registration for lectures take place automatically in Canvas by enabling self-enrollment. This instruction focuses on working with a system in Canvas called "Group sets" (or commonly referred to as "Groups") and can be found in the Canvas course menu 'People'. Although there are many variations to implement, it all boils down to using 2 different methods which can be alternated:


  1. First come, first served   (View example)

    Students click JOIN or LEAVE to register for lectures. The 'First come, first served' method increases the chances of maximum occupancy because (as long as the maximum group size is not yet succeeded) students can register for any lecture provided in the list.

  2. Choose between a number of lectures each time   (View example)

    In this example, students can choose to join one lecture a week (click JOIN, SWITCH, or LEAVE). By using this method we are dividing the opportunity for students to register.

Registration is always open for students. A maximum of 75 places will be set and if this number is exceeded, students can’t register for the particular lecture anymore. However, students can “JOIN” or “LEAVE” lectures, so the list can change until the last moment. If students register for a lecture, we expect them to be there. In case of changes, students need to remove their registration (click LEAVE) as soon as possible to provide a free spot for fellow students.

 


Instructions


1. Go to People in your course menu and select + Group set
 



2. Fill in the right details according to the picture below and click Save
 



3. Notice that you are now navigated to your new Group Set called "Lecture 1"
 

This Group Set is already pre-filled with a group (called "Lecture 1 1", by default). If you can't see any groups, click on + Group in the upper right to create a new group.



4. Navigate to the new group, click on the icon with three dots on the right and select Edit
 



5. Edit the name of this group (with details about the lecture) and click Save
 



6. The first lecture is done!
 

Please note: students can only self-enroll for one group within each Group Set, it is never possible to be part of two different groups within the same Group Set. With that in mind, we are going to create a new Group Set in order for the "first come, first served" method to work. This way, students can enroll themselves to all groups (i.e. lectures).



7. Click on the icon with three dots on the top right and select Clone group set
 



8. Add a new name to this group set and click Submit
 



9. A new Group Set appears (called 'Lecture 2').
  Head over to this new Group Set and edit the details of the lecture (i.e. edit the name of the new group). Repeat the steps above (cloning group sets) as many times as needed to finish up all lectures.



10.

  1. Drag the module to the top of the Modules page (by using the icon with 6 dots on the top left of the module)
    2.

    3.

  4. Now students will automatically be redirected to your registration setup
