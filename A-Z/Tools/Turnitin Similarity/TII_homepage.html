From 26 October 2022, plagiarism checks in Canvas are performed using Turnitin Similarity. This software scans the content for similarities with other sources. On this page, you will be able to find more information about the use of this software.

From Ouriginal to Turnitin Similarity

In October 2022, the UvA has switched from the plagiarism detection software Ouriginal to Turnitin Similarity. On this page, you can find more UvA-wide information on the transition to Turnitin Similarity.

How to use Turnitin Similarity

Curious as to how you can create assignments using Turnitin Similarity? Or how you can interpret a plagiarism report? You can find some more thorough instructions on the Canvas page for UvA instructors.

Suspected plagiarism?

Wondering about what procedure to follow if you suspect a student of plagiarism? This hand-out for teachers will show you how you can view a plagiarism report and how to report suspected plagiarism to the Examinations Boards.
