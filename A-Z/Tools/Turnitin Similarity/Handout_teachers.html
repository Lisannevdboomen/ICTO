When setting up an assignment on Canvas, a plagiarism checker called Turnitin Similarity can be used. This software then generates a percentage and plagiarism report for each file submitted. Teachers can view this report, and based on it, you can decide if you suspect a student of plagiarism. On this page you will find the steps to follow if this is the case.

More information on how to interpret a plagiarism report can be found on the following page.



How can I see a student's plagiarism report?

How do I report suspected plagiarism to the Examination Board?

1.

Go to the relevant assignment on Canvas.
 

2.

Select SpeedGrader in the top right corner and navigate to the student in question.
 

3.

The right field of the SpeedGrader contains the file submitted by the student. Next to it, you'll find the plagiarism percentage. Click on it to view the report.
 

4.

The online plagiarism report opens in a new window.

More information on how to interpret a plagiarism report can be found on the following page.

1.

Open the plagiarism report by following the steps on the left.

2.

In the top right corner of the menu, click the 'Settings' icon to check the settings.
 

3.

Make sure all options below 'Search repositories' are checked.
 

4.

In addition, make sure that no parts of the report are omitted by not checking any boxes below 'Exclude from Similarity Report'.
 

5.

Download the report by choosing Share → Download at the bottom left of the screen.
 

6.

Send an e-mail to the Examinations Board containing [1] the link to the relevant assignment in Canvas and [2] attach the PDF report.

Bij het instellen van een opdracht op Canvas kan gebruik worden gemaakt van een plagiaatchecker genaamd Turnitin Similarity. Deze software genereert vervolgens per ingeleverd bestand een percentage en plagiaatrapport. Dit rapport kunnen docenten bekijken en op basis daarvan kan je beslissen of je een student van plagiaat verdenkt. Op deze pagina vind je meer informatie over de stappen die je doorloopt als dit het geval is.

Heb je vragen over hoe je een plagiaatrapport kunt interpreteren? Ga dan naar deze pagina.



Hoe kan ik het plagiaatrapport van een student bekijken?

Hoe maak ik melding van vermoedelijk plagiaat bij de Examencommissie?

1.

Ga naar de betreffende opdracht op Canvas.
 

2.

Klik rechtsboven op SpeedGrader en navigeer naar de betreffende student.
 

3.

In het rechtervlak van de SpeedGrader staat het bestand dat door de student is ingeleverd. Ernaast staat het plagiaatpercentage. Klik daarop om het rapport in te zien.
 

4.

Het online plagiaatrapport opent in een nieuw venster.

Heb je vragen over hoe je een plagiaatrapport kunt interpreteren? Ga dan naar deze pagina.

1.

Open het plagiaatrapport door de stappen links te volgen.

2.

Klik rechtsboven in het menu op het tandwieltje om de instellingen te controleren.
 

3.

Zorg ervoor dat alle 'Search repositories' staan aangevinkt.
 

4.

En let er daarnaast op dat geen enkel onderdeel uit het rapport wordt weggelaten door geen vinkjes te zetten onder 'Exclude from Similarity Report'.
 

5.

Download het rapport door links onderin het scherm te kiezen voor Share → Download
 

6.

Stuur een e-mail naar de Examencommissie met daarin [1] de link naar de betreffende opdracht en [2] voeg het PDF-rapport als bijlage toe.
