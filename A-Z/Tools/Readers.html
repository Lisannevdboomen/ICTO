A very useful tool within Canvas is Readers. This tool allows you to link all recommended or required literature in one bundle. Such a reading list provides a convenient overview and makes it easy for students to navigate to all publications and PDF files that are required for your course. In addition, it is also possible for students to have the entire reader printed!

There are two ways to create a reader. Wondering which one suits you best? Click here for a schematic overview.


  Quick start

  1 | Make the menu-item Readers visible in your Canvas page

By default, the Readers menu item is hidden in Canvas courses. You can make it visible to yourself and students by following these steps:

  1. Go to the course Settings.



  2. Open the Navigation tab.
    The top menu items are the items visible to students. The lower ones are invisible to students. To make the Readers menu visible, you can drag the item up with your mouse. You can also click the three dots next to the item and click on Enable.



  3. Don't forget to click Save at the bottom of the page.

  2 | Create new list

  1. If you open the tool for the very first time, you’ll see an introductory message in your screen. To create a new list, scroll to the bottom and select the Create it button.



  2. A new screen unfolds. Enter the title and possibly a description of your list here. Click on Create if you have done so.



  3. It is possible to create a blank list or use a template. Choose Default to create a blank list. Sections can be added later, if necessary.



  4. You have now created your first list! Read on to learn how to create sections and add different sources.

  3 | Create sections

  1. If you have created a blank list, there’s automatically one section called Resources. This section can be edited or deleted by clicking the three dots next to the section and selecting Edit section [1] or Delete section [2].



  2. In addition, you can create an entire new section by clicking the button New section [3].

  3. If you choose to edit a section or create a new one, you can enter the title of the section and add a description.



  4. It’s also possible to add a 'from' and 'until' date.
    These dates can be used to solely inform your students, but you can also indicate that the sources in this section should only be visible during the specified period by selecting Section visible only during these dates.



  5. Click Create after you have finished editing the section details.
