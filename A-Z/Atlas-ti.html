What is ATLAS.ti?
ATLAS.ti is a powerful workbench for the qualitative analysis of large bodies of textual, graphical, audio and video data. Sophisticated tools help you to arrange, reassemble, and manage your material in creative, yet systematic ways. ATLAS.ti keeps you focused on the material itself. Whether your field is anthropology, economics, criminology, or medicine: ATLAS.ti will meet your qualitative analysis needs! (https://atlasti.com/product/what-is-atlas-ti/)

UvA free license code
Every year, the UvA shares a new license code which will give students access to Atlas.ti. Please share the following message along with the license code:

"This link is for personal use only and may not be shared with any third party under any circumstances!"

How to install Atlas.ti
Go to this information page and download your Atlas.ti version (Mac or Windows) and install the software on your private computer. You can share this link with your students.

If you are working on a UvA laptop, check the Software Center on a Windows PC (and install Atlas.ti) or contact the Servicedesk ICTS: (020-5251402) servicedesk-icts@uva.nl.

Using Atlas.ti at UvA-location (classroom) 
If you would like to use Atlas.ti in class and want to know if this software is installed on the computers, then please contact the ICTS-servicedesk or visit the location to check.
