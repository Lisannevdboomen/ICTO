1.1. Edit question

1. Go to the Questions menu and select the relevant folder on the left.
 

2. Double-click the question you'd like to edit.
 
 Keyboard shortcuts allow you to perform actions faster. Press [b] to edit a question.

 
  • A.  Question text
  The question text and lay-out can be edited in the field below 'Stem'. The rich content editor provides various options when it comes to the lay-out. Additionally, it is also possible to add links, images, files and other media (see highlighted buttons) here. These will all be visible to students during the exam.
 

 
  • B.  Multiple-choice options
  You can add various answer options in the field below 'Options'. To add answer options, click on 'Options' and select + or press enter on your keyboard [1]. You can then edit the answer text. After having added multiple answer options, you will need to select the correct answer [2].
 

 
  • C.  Grading criteria open questions
  If you want to use open-ended questions, you will always have to fill in grading criteria at the bottom of the page. You can also use multiple criteria (a check-up model) for subsections that make up an essay question. To add criteria, click â€˜Grading criteriaâ€™ and select + or press enter [1]. It is possible to edit the name [2], the number of attainable points [3], and the criterion text [4].

 

4. Donâ€™t forget to click 'Save' and publish the test.
  Be sure to publish the test each time you edit a question so that these changes are also visible to students when they make the exam.

1.2. Edit question properties

1. Go to the Questions menu and select the relevant folder on the left.
 

2. Double-click the question you'd like to edit and click 'Properties' in the top right corner.
 

 
  • A.  Number of points
  Choose how many points can be obtained for a correct answer to a multiple choice question.
 

 
  • B.  Random order
  Present answer options to a multiple choice question in randomized order.
 

 
  • C.  Word count
  Set the maximum number of words a student may use to answer an open-ended question.
 

 
 Note: students are able to see the word count and are alerted if they exceed it. However, they can always continue typing. In other words, it functions as an indicator rather than a limit.

2.1. Review test settings

1. Go to the Tests menu and select the relevant folder on the left.
 

2. Double-click the exam of which you'd like to review the settings.
  The exam has already been created for you by ICTO, so the only thing youâ€™ll need to do is preview and check its settings. We advise you to pay extra attention to the following points:
  • Instruction text
  • Question blocks
  • Randomize questions
  • Number of points
  • Test properties
The instruction text has been copied from last year's test. Make sure the contents are still relevant and accurate (e.g., check whether dates are still correct). By default, there's no result text added to the exam.
The use of question blocks is based on last year's exam. Check whether or not you'd like to use question blocks [1] this year, and decide whether you'd like to present them in a randomized order [2].
Besides random block order, it is also possible to present the questions themselves in a random order. If no questions blocks are used, the option 'Random question order' will be visible at the top of the page. If they are used, you can edit each block (via de pencil icon besides each block) and check whether questions are presented in a random order or not.

It is useful to check whether the number of points that can be obtained is correct. You can see the number of points next to each question in the test, and the total number of points can be found in the bottom left corner.

2.2. Publish test

1. Go to the Tests menu and select the relevant folder on the left.
 

2. Check the publication status next to the relevant test.
  If changes have been made without publishing the test, the status is set to 'Not synchronised', which means students will not be able to see the latest version of the test.
 

3. If the publication status is set to 'Not synchronised', click the arrow next to the test to publish it.
 

4. A new screen will pop up. In this screen, you'll be able to see which changes were made. Click Publish in the lower right corner.

Create new questions

 
Please note that, as a teacher, you are not required to create new questions. The ICTO team prepares dummy questions for you based on last year's exam and ensures these questions are imported in the test. This means you are only required to modify these questions. The instruction below only applies to you if you want to add extra questions. Beware that you still need to add these questions to the test. Otherwise, students will not be able to see your newly created questions at the test.

1. Go to the Questions menu and select the relevant folder on the left.
 

2. Select 'New' at the top of the page.
 

 
 TestVision has several keyboard shortcuts that allow you to perform actions faster. Press [n] to create a new question.

3. Modify the question to your liking.
 
 Instructions on how to edit a question are linked below:
      → To modify the question type, click here.
      → To add question text, answer options and grading criteria, click here.
      → To edit points, randomize answer options and add word count, click here.

4. Click Save once you're done editing a question.
 

5. Enter a question name [1] and adjust the question status by clicking Change question status [2].
  Several options will appear; to ensure the question will be visible to your students, the question status needs to be set to 'Approved'.
 

 

 Do you get an error message when you want to change the question status?

Check whether your question meets all requirements. For instance, it can happen that you haven't filled in the answer options or haven't selected the correct answer option yet. And with open questions, it is required that you add grading criteria.

If you cannot figure out why you cannot change the question status, please send us an email via icto-csw@uva.nl.
5. Add the questions to the test.

Add questions to test

 
Please note that, as a teacher, you are not required to add questions to the test if you use the dummy questions the ICTO team has prepared for you. The instruction below only applies to you if you have decided to take on the task of creating new questions yourself.

1. Go to the Tests menu and select the relevant folder on the left.
 

2. Double-click the test in which you'd like to add new questions and select '+ Questions' in the left screen.
 

 
 A copy from last year's test has already been prepared for you, and dummy-questions have been added to this test. Therefore, there's no need to create a new test. Newly created questions, on the other hand, aren't automatically added to this test. You'll need to add them to the test yourself.

3. Navigate to the relevant question folder.
4. Select the questions you'd like to add to the exam and move them to the question set of the exam.
  → Select all questions [1a] or each question separately [1b].
→ Move all questions to the question set [2a] or move each question separately [2b].
 

5. If you have moved all questions to the question set, click 'Close' at the bottom of the screen.
  The questions you selected are now in the question set of the test: they are not yet in the test itself.

 

6. Move the questions to the test (the right screen in the image below) by clicking the arrows next to the questions [1].
  The checkmarks [2] in front of the questions indicate whether a particular question is also in the test.

 

 
 Are you unable to add a question? This is likely caused by the fact that the question status is not set to 'Approved' yet. For more info, click here and navigate to step 4.

4. Donâ€™t forget to publish the test.
  Be sure to publish the test each time you make adjustments to the test so that these changes are also visible to students when they make the exam.

Edit test settings

 
Please note that, as a teacher, you are not required to edit test settings, as you can also ask ICTO to do this for you. However, if you'd like to take on this task yourself, you can use the guidelines below to do so. This is particularly useful if during reviewing the test you ran into things that needed to be adjusted or if you created and imported new questions.

1. Go to the Tests menu and select the relevant folder on the left.
 

2. Double-click the test you'd like to make changes to.
  • A.  Instruction text
  Go to the 'Instruction text' tab and select the text that's already there. If there's no text yet, click the button next to 'Own text' or press [t].
 

  • B.  Question order
  Go to the 'Test' tab and press the on/off switch next to 'Random question order' if you'd like to present the questions in randomized order [1]. If you're using a fixed order, you can move the questions around by clicking the the three bars on the right of each question [2]. Hold the button while you drag the question to the place of your preference.

 

 
  • C.  Add & randomize blocks
  It's possible to make use of question blocks to divide your questions. To do so, you'lff first have to press the on-/off switch next to 'Blocks' [1]. After this you can add blocks [2] and decide to present these blocks in a random order [3]. This doesn't have anything to do with the order of the questions itself.

 

 
  • D.  Move blocks
  If you present the blocks in a fixed order, it is also possible to move the blocks around. You can do so by clicking the arrows on the left of each block.

 

 
  • E.  Move questions in blocks
  You can move questions following the guidelines mentioned in part B 'Question order'.
 
  • F.  Block properties
  Edit block properties by clicking the little pencil icon on the right side of each block.
 

  You can decide to make the block introduction visible for students [1]. In this introduction you can add instructions relevant to that question block [3]. It's also possible to present the questions inside this block in a random order [2]. This can be adjusted per block.

 

  • G.  Test properties - general
  On the right, you can adjust the test properties. Among other things, you can adjust the maximum test time, which always refers to the 'regular' test time. It is also possible to decide that questions or blocks will only show once. The ICTO strongly advises against this option, as students will then not be able to browse through the questions. You can also add digital scrap paper.

 

  • H.  Test properties - grading
  Grading properties can be found below the regular test properties on the right side. This part is set in a standard way by ICTO. This standard can be seen in the image below. Naturally, this standard may be modified. This can also be done after students have taken the test.

 

  As you can see in the standard grading settings shown in the image above, a chance score is applied to all multiple-choice questions. Additionally, grades are always higher than 1, and at a score of 55%, students receive a 5,5 as a grade.
