Try to reduce the risk of fraud


An online exam requires knowledge of a higher order
  •  

    Unaudited, online exams are by definition open book. Nothing prevents students from using resources such as the Internet, textbooks and notes. Your exam must be designed for this. In an unshielded testing environment, the student can share screenshots via whatsapp and other social media, or google. An online exam requires knowledge of a higher order (deeper understanding, application, analyzing, evaluating) and is preferably strictly timed, which makes collaboration and plagiarism unattractive. (source).

Here are some practical tips!

  •  
    1. Use a strict time limit at the exam
    2. Offer questions in a random order
    3. Offer answer options in a random order
    4. Show one question at a time
    5. Students are only allowed to answer the question once, block the question after it has been answered
    6. Make each test unique: create a larger pool of multiple choice questions (e.g. 50 questions in a Question Bank) and let students answer a random combination of these (e.g. 25 questions)
    7. The Question Banks are also important in order to cover all course content in an exam. If you have 4 main topics, then you create 4 Question Banks, so that each student gets the same number of questions about each main topic.
